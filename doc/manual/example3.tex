\section{Пример III}
Задана ЗЛП:
$$F(\vec{X}) = x_1+4x_2 \to max$$
\begin{equation}
\label{system}
\begin{cases}
2x_1+4x_2 \le 17\\
10x_1+3x_2 \le 15\\
x_i \ge 0 \\
\end{cases}
\end{equation}

\subsection{Решение исходной ЗЛП}
Введем искусственные переменные и приведем к каноническому виду.
Для нахождения максимума, умножим целевую функцию на -1.

$$-F(\vec{X}) = -(-x_1-4x_2) \to max$$
\begin{equation}
\label{cannonical}
\begin{cases}
2x_1+4x_2+x_3=17\\
10x_1+3x_2+x_4=15\\
x_i, s_i \ge 0 \\
\end{cases}
\end{equation}
